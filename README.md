#  Create a new repository on the command line

### Git Initialized
**`git init`**

### Add To Git
**`git add README.md`**
or all 
**`git add .`**

### Git Commit
**`git commit -m "first commit"`**

### Git Set Branch
**`git branch -M main`**

### Git Add Origion
**`git remote add origin https://github.com/wapborhan/wapborhan-next-jul-24.git`**

### Git Branch wise Push
**`git push -u origin main`**
